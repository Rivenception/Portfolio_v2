Software I use, gadgets I love, and other things I recommend.


I get asked a lot about the things I use to build software, stay productive, or buy to fool myself into thinking I’m being productive when I’m really just procrastinating. Here’s a big list of all of my favorite stuff.
Workstation
ACER Nitro AN515-46 Laptop
My primary workstation is the ACER Nitro AN515-46, a high-performance laptop built for both productivity and gaming. It offers the power and speed I need for handling intensive tasks like software development, data analysis, and gaming, all in a portable package.

AMD Ryzen 7 6800H Processor with Radeon Graphics
Powered by an AMD Ryzen 7 6800H processor running at 3.20 GHz, this chip gives me the speed and multi-threaded performance needed for complex tasks and multitasking. Its integrated Radeon graphics further enhance its ability to handle graphics-heavy applications smoothly.

NVIDIA GeForce RTX 3070 Ti Laptop GPU
With the NVIDIA GeForce RTX 3070 Ti GPU, my workstation is equipped to handle advanced graphic processing for gaming, rendering, and machine learning applications. It provides excellent performance for graphics-intensive projects and ensures smooth visuals.

Micron 3400 NVMe SSD
The Micron 3400 NVMe SSD offers fast read and write speeds, allowing for quick boot times and fast file transfers. This significantly improves overall system responsiveness, making my workflow much more efficient.

RAM 32GB
With 32 GB of installed RAM, my laptop easily handles multiple programs and large datasets without any lag. Whether I’m running virtual machines or working with data-heavy applications, the system has more than enough memory to keep up.

Windows 11 64-bit OS
Running on a 64-bit version of Windows 11, my system is optimized for modern software and applications. This setup provides full access to the RAM and ensures compatibility with the latest software and technology.


Development Tools
VS Code
A powerful and customizable code editor. It supports a wide range of programming languages and extensions, making it ideal for both frontend and backend development. It’s my go-to tool for writing, debugging, and version controlling code.

Git & HitHub
Git is a version control system that tracks changes in code, while GitHub is a platform for hosting and collaborating on Git repositories. I use them to manage my code, collaborate with others, and keep track of project versions efficiently.

Node.js
Node.js is a runtime environment that allows me to run JavaScript on the server side. It’s highly scalable and perfect for building fast, efficient applications, especially when handling numerous simultaneous requests. I use it for backend development and real-time applications like chat services or APIs.

MySQL
MySQL is a widely-used relational database management system. I use it for building and managing databases in my projects, ensuring data is stored efficiently and queries are processed quickly. It’s perfect for handling structured data in web and software applications.

Postman
Postman is a platform used for testing APIs. I use it to send requests to web services and test responses, making sure that everything works smoothly before integrating APIs into larger projects.

Docker
Docker is a tool that allows for creating, deploying, and running applications inside containers. It simplifies the process of setting up consistent development environments across different machines and systems.

Design
Figma
Figma is a cloud-based design tool used for UI/UX design and prototyping. It allows for real-time collaboration, making it easy for teams to work on design projects together, whether they're working on wireframes, prototypes, or final UI designs.

Tableau
Tableau is a powerful data visualization tool. I use it to create interactive, shareable dashboards and reports, helping to turn complex data sets into easy-to-understand visual stories. It’s essential when I need to present data insights to non-technical audiences in a clear and engaging way.

Productivity
Slack
Slack is a communication tool used for team collaboration. I use it to chat with coworkers, share files, and integrate other productivity tools to keep everything in one place.

Google Colab
Google Colab is a cloud-based Jupyter notebook environment that I use for running Python code and performing data analysis. It’s especially useful when working with machine learning models and data science projects, as it provides free access to GPUs and easy sharing with collaborators.

Microsoft Teams
Microsoft Teams is my go-to tool for team communication and collaboration. I use it to chat, hold video calls, and collaborate on files with colleagues, all within a seamless interface. It’s an essential tool for managing remote teams and staying connected.

Smartsheet
Smartsheet is a work management platform that combines project management, automation, and collaboration in one place. I use it for tracking tasks, deadlines, and project timelines, ensuring that everything runs smoothly and on schedule. It’s a great tool for managing projects at scale, especially with teams.

Discord
Discord is a communication platform primarily used for voice, video, and text chat. I use it for collaborating on projects with peers, participating in coding communities, and even for team coordination in informal settings. It’s great for real-time communication, especially in development or gaming communities.
